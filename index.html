A simple text-based electronics simulator



BRU uses its own flavor of HDL(hardware description language) called Brudl and a minimalistic scripting language named Brusc to design and simulate digital electronics components, like logic gates and chips.

To use the BRU tool, clone the git repository or download the Go file [bru.go]. Build an executable for your system configuration using the Go build tool as follows :

go build bru.go

Once the executable is ready, write and save your hdl code and your script. The run the tool with your files as follows :

(On Windows)

bru.exe path-to-hdl-file -s path-to-script

What now? Well, go bananas!

Bru is free software, so you can modify the code and experiment with it all you want. If you want a particular feature to be added to a future release, drop me an email at dev.godboleashwin@gmail.com. If you discover a bug, open an issue on github and I'll try and fix it as soon as possible.
